Для запуска приложения требуется:
1) перенести файл базы данных по пути ../EXE/playersandteamstatistic.mdf на SQL Server
2) Запустить проект из папки PRG
3) Скопировать ссылку из адресной строки
4) заменить в файле e_x_e.bat ссылку на скопированную


В случае неисправностей:
1) проверить строку подключения в файле Web.Config по пути ../PRG/StatTrack/Web.Config
2) заменить значение "catalog" на ваш
2.1) В обозревателе серверов можно проверить подключение к бд, если оно неудачно, создать новое подключение и скопировать оттуда строку подключения
